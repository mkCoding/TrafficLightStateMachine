# Description
- Project allowing users to switch between different traffic light colors
- Simple project to help me understand State Machines. 

# Goal
- Project must: define clearly defined states - **Sealed Class**
- Project must: determine exactly how states move - **ViewModel**
- Project must: have UI that reacts to State change - **UI Screen**



> State Machine: a project can be in only 1 state at a time based on current state or specific defined rules
